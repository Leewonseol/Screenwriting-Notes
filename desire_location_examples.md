# 시나리오에서 장소, 욕망, 행동의 원칙

| 원칙 | 내용 |
|------|------|
| **원칙 1** | **욕망이 먼저다.** 장소는 욕망을 실행하기 위한 선택지다. |
| **원칙 2** | **장소는 인물의 욕망에 대한 라벨이다.** 단순한 지명이 아니라 어떤 명칭과 수식어가 욕망을 잘 드러내는지 주의해야 한다. |
| **원칙 3** | **장소는 정보 1개만 준다.** (수식어 1개면 끝) |
| **원칙 4** | **욕망은 결과가 아니다.** 지금 그냥 두지 못하는 관계 반응이다. |
| **원칙 5** | **행동은 욕망의 수단이다.** 목적·의미·결과 쓰지 않는다. |

# <굿 윌 헌팅>에서 욕망이 장소 명칭에 반영된 예시

| 욕망 (지금 그냥 두지 못하는 것) | 장소 선택 | 왜 이 장소가 맞는가 |
|--------------------------------|-----------|---------------------|
| **내 사람을 무시당하지 않게 하고 싶다** | **번잡한 술집** | 끼어들기·충돌·말싸움이 자연스럽다 |
| **상대의 재능을 가지고 싶다** | **교수 연구실** | 위계가 철폐된다. 정식 랩실은 위계 그 자체인데, 교수가 자신의 연구실로 낮선 사람을 초대했다. |
| **통제를 회피하고 싶다** | **조용한 상담실** | 규칙이 있는 공간일수록 회피/도발이 드러난다 |
| **상처를 드러내지 않고 버티고 싶다** | **상담실(대면)** | 침묵·저항이 행동으로 보인다 |
